detail_/utilities/repeat_string.cmake¶

detail_/utilities/repeat_string.cmake._ct_repeat_string(_rs_result _rs_str _rs_n)¶

Creates a string by repeating a substring

This function is used to create a new string by repeating a substring a

specified number of times. This is useful for creating the proper indents

for printing and for creating repeated characters like in banners.

Parameters

  • _rs_result (Identifier) – An identifier to save the result to.

  • _rs_str (str) – The substring to repeat.

  • _rs_n (str) – The number of times to repeat ${_rs_str}.

Returns

A string containing the created string. The result is accessible to

the caller via _rs_result.
