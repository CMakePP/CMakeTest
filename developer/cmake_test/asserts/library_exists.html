cmake_test/asserts/library_exists.ct_library_target_exists(_lte_result _lte_name)

Determines if a target exists and if it is a library.

This function checks whether a target exists and then checks whether that target is of type STATIC_LIBRARY, MODULE_LIBRARY, or SHARED_LIBRARY and then returns an integer indicating what it found.

Parameters

  • _lte_result (int) – Name to use for the variable which will hold the result.

  • _lte_name (string) – The target name to check.

Returns

_lte_result will be set to 0 if the target is a library, 1 if the target is not a library, and 2 if target does not exist.

Return type

int
