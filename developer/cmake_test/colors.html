cmake_test/colors

This module provides an option called CMAKETEST_USE_COLORS and a set of global variables to use to color terminal output. These global constants are just one-to-one constants of specific ANSI control codes. If CMAKETEST_USE_COLORS is not set to true or if the OS is Windows (cmd doesn’t support ANSI control codes), these constants will not be defined and should resolve to an empty string.

Example:

cmake_test/colors.string(ASCII 27 Esc)

Warning

This is a generic command invocation. It is not a function or macro definition.

The character used to start the ANSI control code.
