asserts/file_contains.cmake¶

asserts/file_contains.cmake.ct_assert_file_contains(_afc_file _afc_text)¶

Assert a file contains the specified text.

Asserts that the file at the specified path contains the specified text.

Will return if the file does not exist, the assertion failure will be logged as a fatal error.

Parameters

  • _afc_file (path) – The file to check

  • _afc_text (string) – The text to check for

asserts/file_contains.cmake.ct_assert_file_does_not_contain(_afdnc_file _afdnc_text)¶

Assert a file does not contain the specified text.

Parameters

  • _afdnc_file (path) – The file to check

  • _afdnc_text (string) – The text to check for

asserts/file_contains.cmake.ct_file_contains(_fc_result _fc_file _fc_text)¶

Determines if a file contains some text.

This function checks whether a file contains some text and returns a boolean

result.

Will return if the file does not exist, the assertion failure will be logged as a fatal error.

Parameters

  • _fc_result (bool) – Name to use for the variable which will hold the result.

  • _fc_file (string) – The file to check.

  • _fc_text (string) – The text to check for.

Returns

_fc_result will be set to TRUE if file contains the text

and FALSE if it does not.

Return type

bool
