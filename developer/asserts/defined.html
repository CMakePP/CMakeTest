asserts/defined.cmake¶

asserts/defined.cmake.ct_assert_defined(_ad_var)¶
Asserts that the provided variable is defined.

This function can be used to assert that a variable is defined. It does not assert that the variable is set to any particular value. If the variable is not defined it will raise an error.

Parameters

_ad_var (Identifier) – The identifier to check for defined-ness.

asserts/defined.cmake.ct_assert_not_defined(_and_var)¶
Asserts that the provided variable is not defined.

This function can be used to assert that a variable is not defined. If the variable is actually defined this function will raise an error.

Parameters

_and_var (Identifier) – The identifier to check for defined-ness.
